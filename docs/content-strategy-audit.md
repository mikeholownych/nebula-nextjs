# Content Strategy Audit — Against Duncan Rogoff's "Teach vs Flex" Insight

## The Insight

Duncan's Claude Code analysis of 1,546 posts found:
1. **Teaching beginners** → attracted buyers
2. **Flexing about replacing team** → buyers vanished, competitors filled comments
3. Most frequent commenters ≠ ideal clients

## Nebula Content Audit

### What We're Doing Right ✓

| Asset | Posture | Verdict |
|-------|---------|---------|
| **Site headline** | "Your landing page is leaking buyers" | TEACH — names the problem, offers solution |
| **Site copy** | "For founders running paid traffic with weak conversions" | TEACH — audience-specific, not flexing |
| **Cold email templates** | "ran a landing page audit for {{company}}" | TEACH — value-first, audit hook |
| **Email sequences** | "What the top 1% of landing pages do differently" | TEACH — teaching, not showing off |
| **LinkedIn replies** | We respond with value, not "I built this amazing thing" | TEACH — verified by our reply templates |
| **Beta tester page** | "You get the Fix Pack free in exchange for feedback" | TEACH — collaborative, not transactional |
| **No-calendar CTA** | Never forced "book a meeting" | TEACH — self-serve before sales |

### No Flexing Found ✓

Audited all public-facing copy: site, checkout, email templates, email sequences, audit delivery, beta signup. No "I replaced my team with AI" or "Look how smart I am" language. Zero.

### What We Could Do Better

| Gap | Fix |
|-----|-----|
| **No "teach beginners" LinkedIn content yet** | Our first post is on the audit tool itself. Need a "How to fix a landing page in 15 minutes" teach thread. |
| **Site has testimonial blocks with no real customers** | Duncan: "fake trust or no trust" — we should rotate to beta tester case studies ASAP |
| **No content angle taxonomy** | We track which post → which lead, but don't classify angle. Added in lead_manager.content_posts.angle field. |
| **No content performance dashboard** | Added /api/content-performance endpoint (auth-protected). Connect to dashboard. |

### Recommendation

Maintain the TEACH posture across all channels. The one risk area is if we start doing "we're so automated" flexing instead of "here's how to fix your page" teaching. If we ever publish "I built an AI that runs my entire business without me," that's the flex that loses buyers.

**Action:** First LinkedIn post or thread should be "How to find the 5 leaks in your landing page in 60 seconds" — pure teaching, zero flex.
