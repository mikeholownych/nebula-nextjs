# Nebula Customer Portal

Next.js frontend for the Nebula Components Platform API.

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)

## Environment Variables

Create `.env.local`:

```bash
NEXT_PUBLIC_API_URL=http://localhost:8769
```

## Features

- Google OAuth login
- Session management
- User dashboard
- Organization view

## Tech Stack

- Next.js 16.2.10
- TypeScript
- Tailwind CSS
- @react-oauth/google
