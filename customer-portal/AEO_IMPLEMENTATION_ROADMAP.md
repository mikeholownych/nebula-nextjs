## Phase 5: Run Controlled Measurement (Week 7)

**Objective:** Citation tracking, fidelity review.

| Metric | Tool | Cadence |
|--------|------|---------|
| Citation presence | Manual/PromptLoop | Weekly |
| ChatGPT referrals | GA4 | Daily |
| Bing AI Performance | Bing Webmaster | Weekly |
| Answer fidelity review | Manual | Monthly |

**Testing Protocol:**
- Run top 20 queries across Google AI, ChatGPT, Perplexity, Bing
- Track: cited domain, cited URL, claim fidelity
- Repeat from clean accounts
- Record timestamp, locale, model

**Deliverables:**
- Citation tracking spreadsheet
- Answer fidelity report
- Competitor citation analysis

## Phase 6: Scale Proven Patterns (Weeks 8-12)

**Objective:** Expand what works.

Scale only patterns that demonstrate:
- Increased citation share
- Accurate attribution
- Qualified referral or assisted conversion

**Expansion Criteria:**
- If definition pages cite: build 20 more
- If comparison pages cite: build 10 more
- If case studies drive referrals: build 100 more

**Governance Implementation:**
- Content owner assignment (all pages)
- Review date tracking
- Change history log
- Incident playbook (misquotation response)
