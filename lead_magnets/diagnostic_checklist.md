# Landing Page Diagnostic Checklist

## 1. Hero Section
- [ ] Clear value proposition above the fold
- [ ] Benefit-focused headline (not feature-focused)
- [ ] Strong CTA button with action-oriented text
- [ ] Social proof/testimonial visible

## 2. Problem Agitation
- [ ] Clearly states the pain point your audience feels
- [ ] Uses specific, relatable language
- [ ] Shows understanding of their frustration

## 3. Solution Presentation
- [ ] Explains how your offer solves the problem
- [ ] Focuses on outcomes, not features
- [ ] Uses visual aids to demonstrate value

## 4. Social Proof
- [ ] Customer testimonials with photos/names
- [ ] Trust badges/logos
- [ ] Case studies with measurable results

## 5. CTA Optimization
- [ ] Clear next step with low friction
- [ ] Multiple CTAs throughout the page
- [ ] Urgency or scarcity element (if appropriate)

## 6. Mobile Responsiveness
- [ ] All elements render properly on mobile
- [ ] Font sizes are readable
- [ ] CTAs are thumb-friendly

## 7. Load Speed
- [ ] Page loads in under 3 seconds
- [ ] Images are optimized
- [ ] Unnecessary scripts are minimized

## 8. Conversion Elements
- [ ] Form fields are minimized
- [ ] Security indicators present
- [ ] Clear privacy policy link

## 9. SEO Basics
- [ ] Page title optimized for target keyword
- [ ] Meta description compelling
- [ ] Heading structure logical (H1-H6)

## 10. Trust Factors
- [ ] Guarantee/warranty mentioned
- [ ] Contact information visible
- [ ] Physical address if applicable
