# Lead Magnet Strategy

## Core Purpose
To solve a specific, painful symptom the prospect has right now while logically positioning the Core Offer as the only viable solution to the root cause.

## Three Prompt System
1. **Bridge Asset Strategist**: Analyze ICP Memo and Offer Memo to generate 3 lead magnet concepts
2. **Deep-Dive Interview Protocol**: Extract specific expertise through voice recording and transcription
3. **Ghostwriter Loop**: Write the asset one section at a time with feedback integration

## Implementation Steps
1. **Strategy Phase**: Pick strongest concept from AI-generated options
2. **Extraction Phase**: Record voice notes answering specific expertise questions
3. **Architecture Phase**: Build psychological blueprint for chapter-by-chapter persuasion
4. **Production Phase**: Write using ghostwriter loop with previous text feedback
5. **Polish Phase**: Review and publish final draft

## Output Format
3,000-10,000 word document distributed through:
- LinkedIn posts
- Email sequences
- Outreach DMs

## Success Metrics
- **Reply Rate**: Target 3-5% from cold outreach
- **Pipeline Impact**: Revenue attributed from UTM-tagged links
- **Deliverability Rate**: 90-98% inbox placement
