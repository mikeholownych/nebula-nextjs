# Hormozi-Style Communication Principles

## Core Tenets
1. **Ultra-Concise**: Every message should be under 80 words
2. **Value-First**: Lead with concrete solutions, not generic pitches
3. **Problem-Focused**: Reference specific pain points from the ICP Memo
4. **Risk Reversal**: Include strong guarantees and proof elements
5. **Single CTA**: One clear next step per message

## Implementation Rules
- **Subject Lines**: Use curiosity gaps and problem callouts
- **Opening Lines**: Personalize with specific context
- **Body Copy**: Value-first, specific, outcome-focused
- **Offer Structure**: Price anchoring with strikethrough original prices
- **Risk Reversal**: "Ship or I Pay You" guarantee format
- **CTA Format**: Direct link to sales page with UTM parameters

## Prohibited Language
- "leverage", "harness", "unlock", "unveil", "delve", "underscore"
- Any hedging language like "might" or "could"
- Generic marketing phrases without specific context
