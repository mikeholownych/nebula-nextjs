# Offer Architect Framework

## Core Principles
1. **Radical Offer Surgery**: Reconstruct commoditized offers using the Grand Slam Value Equation
2. **Value Equation**: Maximize Dream Outcome and Perceived Likelihood, minimize Time Delay and Effort
3. **Risk Reversal**: Neutralize silent objections that kill deals
4. **Quick Wins**: Deliver value within first 48 hours

## Implementation Steps
1. **Dream Outcome Upgrade**: Use market language from ICP Memo
2. **Likelihood Maximizer**: Add proof, guarantees, and case studies
3. **Time Compression**: Design early wins for first week
4. **Effort Reduction**: Eliminate, automate, or do-for-them high-friction steps

## Output Structure
1. **New Promise**: One sentence in market language
2. **What Included**: Deliverables reframed as outcomes
3. **Risk Reversal**: How silent objection is neutralized
4. **Quick Win**: First 48-hour result
5. **Effort Map**: Client actions vs. done-for-you steps
6. **Pricing Logic**: Value equation alignment check

## Quality Gates
- Does "New Promise" use bleeding neck verbatim from ICP Memo?
- Does "Risk Reversal" address the #1 unstated reason prospects go quiet?
- Is "Quick Win" something you can actually deliver in first week?
