# Cold Outreach Doctrine
> Trigger-aware only. Build the artifact before you ask. No spray-and-pray.

## Core Rule
Never contact someone without a specific trigger that proves they have the pain RIGHT NOW.

## The Trigger Stack (ranked by signal quality)
1. **Job post signal** — posting for "CRO", "landing page", "paid ads help" on Upwork/LinkedIn
2. **Forum complaint** — r/PPC, r/Entrepreneur: "my ads aren't converting"
3. **Ad launch signal** — LinkedIn post announcing new campaign (pain arrives in 30 days)
4. **Agency frustration** — complaining about their current agency

## Outreach Formula
1. Find the trigger
2. Build the artifact (run their audit, show the score)
3. Send the artifact as the first message — NOT a pitch
4. CTA = "here's what I found, want the full fix list?"

## What NOT to do
- Generic "I help founders improve conversions" opener
- Filter-based list (industry + headcount + title) with no trigger
- Ask for a call before delivering value
- Reply-gate ("reply YES to get your audit")

## Email Tooling
- AgentMail: REST only (no SMTP). 403 = suppressed address.
- Resend: transactional only (hello@nebulacomponents.shop)
- Cold email: separate sending infrastructure needed for scale

## Channels
- Upwork: apify/neatrat~upwork-job-scraper (RSS dead Aug 2024)
- LinkedIn: xurl CLI or manual
- Reddit: search-based, manual DM
