# Nebula Value-First Outreach Rule

Source lesson: founders hate unsolicited marketing vomit. Generic claims like “I can generate leads” destroy trust.

## Rule

Do not lead with a pitch. Lead with a useful artifact.

## Operating standard

1. Hand-pick prospects with a visible buying trigger.
2. Build or generate something useful before asking for anything:
   - landing-page leak note
   - mini audit
   - teardown screenshot
   - SOP/checklist
   - first-fix implementation sketch
3. Send the artifact with a no-pressure opener:
   - “I made this in case it helps.”
   - “No ask attached.”
   - “Keep the output either way.”
4. Only offer paid implementation after the prospect has received value or clicked into the self-serve audit.
5. Never use generic positioning:
   - “I generate leads”
   - “marketing expert”
   - “growth guru”
   - “book a call”
   - “worth a look?”

## Default CTA

Use this before any checkout CTA:

```text
No ask attached. If it helps, run it here and keep the output either way:
https://nebulacomponents.shop/audit.html
```

## Revenue logic

This is not softer. It is higher-congruence selling:

trigger evidence → useful artifact → self-serve audit → $97 implementation.
