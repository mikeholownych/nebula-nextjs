# Nebula Team Playbook

## Core Principles
1. **Structured Process Over Genius** - Follow the framework, not individual brilliance
2. **Source-Driven Insights** - Every recommendation must trace back to data
3. **Readiness First** - Never recommend what's not feasible
4. **Financial Validation** - All solutions must prove their worth
5. **Phased Execution** - Build confidence through early wins

## Daily Workflow

### Morning Standup (15 mins)
- Review yesterday's progress
- Identify blockers
- Assign daily priorities

### Discovery Phase Protocol
**Discovery Day 1-2:**
- Complete discovery template within 48 hours
- Conduct all stakeholder interviews within 72 hours
- Complete readiness assessment by end of Day 2

**Discovery Day 3:**
- Synthesize findings
- Identify key opportunities
- Prepare for solution phase

### Solution Phase Protocol
**Solution Day 1:**
- Map opportunities to readiness levels
- Develop financial models for top 3 opportunities
- Create phased roadmap

**Solution Day 2:**
- Validate recommendations with client
- Adjust based on feedback
- Finalize deliverable

### Delivery Phase Protocol
**Delivery Day 1:**
- Present discovery findings with source attribution
- Share readiness assessment
- Present financial models

**Delivery Day 2:**
- Present phased roadmap
- Get client buy-in on quick wins
- Schedule implementation kickoff

## Quality Gates

### Discovery Gate
- All sources documented
- Readiness assessment completed
- Key opportunities identified

### Solution Gate
- Financial models validated
- Recommendations ranked by impact/effort
- Roadmap dependencies mapped

### Delivery Gate
- Client buy-in secured
- Implementation timeline agreed
- Success metrics defined
